# 🏥 AI Healthcare Claims Intelligence System

An AI-powered healthcare claims platform built using FastAPI, NVIDIA LLMs, and Retrieval-Augmented Generation (RAG).

## Features

- 🤖 NVIDIA Llama Integration
- ⚡ FastAPI Backend
- 📄 PDF Processing
- 🔍 RAG Pipeline (Coming Soon)
- 🏥 Healthcare Claim Analysis
- 📊 Fraud Detection (Coming Soon)
